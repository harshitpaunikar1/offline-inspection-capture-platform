# Offline Inspection Capture Platform

> **Domain:** Logistics

## Overview

Lightweight field inspection app with in-app camera, mandatory field validation, predefined checklists, and queued uploads with retry logic for low-spec devices.

## Skills & Technologies

- Android Development
- Offline-First Architecture
- SQLite Data Storage
- Background Sync
- Form Design & Validation
